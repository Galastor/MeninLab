Чтобы играть, в файле meninlab.lisp в первой строке пропишите папку, в которой находится игра. Загрузите файл meninlab.lisp в common lisp.

---------------Изменения-------------
Коммит от 25.09.2013
+Теперь в игру можно играть, не только смотреть
+Взаимодействие с игрой идет с помощью паттерна "команда"
+Удар проходит всегда, без воздействия рандома
+Возможность убежать из боя (при этом теряется опыт)
+Возможность посмотреть информацию о персонаже и противнике
