Чтобы играть, запустите файл meninlab.exe
Этот файл больше не будет меняться (по крайней мере не должен...)
При обновлении достаточно скачать новые исходники в папку с игрой.
Этот файл по сути sbcl, загружающий meninlab.lisp, который в свою очередь загружает исходники и запускает игру.
